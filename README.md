# Planned Features

# Requirements for compiling

To compile the project you will need:

1. Unreal Engine 4.25
2. File System Library ( https://www.unrealengine.com/marketplace/en-US/product/file-system-library )
3. JSON Blueprint ( https://www.unrealengine.com/marketplace/en-US/product/json-blueprint )
